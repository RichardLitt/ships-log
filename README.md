# ships-log

> Open up a log file for today in a project folder

## Install

```sh
npm i -g project-logg
```

## Usage

```sh
log
# Will automatically open up today's date file in log/

# Or, to see yesterday's tasks
log --yesterday
# Will print them to stdout
```
